# Tomorrow - Estrutura de Dados

Este repositório contém exercícios e desafios sobre estruturas de dados em C++, além de uma interface web interativa de Perguntas e Respostas.

## 📚 Estrutura do Repositório

- **Challenges/** - Desafios de programação organizados por dificuldade
  - WorthPoints/ - Desafios que valem pontos
  - NoWorthPoints/ - Desafios sem pontuação
- **Class1/** - Exemplos da primeira aula
- **Class2/** - Exemplos da segunda aula

## 🌐 Interface Web - Perguntas e Respostas

Uma página web interativa para visualizar e adicionar perguntas sobre estruturas de dados.

### Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Visualize as perguntas e respostas sobre estruturas de dados
3. Clique no botão "Adicionar Nova Pergunta" para adicionar dinamicamente novas perguntas

### Características

- Design moderno com gradientes em roxo
- Animações suaves e responsivas
- Botão interativo na parte inferior da seção de perguntas
- Notificações de sucesso ao adicionar perguntas
- Layout responsivo para dispositivos móveis

### Tecnologias Utilizadas

- HTML5
- CSS3 (com animações e gradientes)
- JavaScript (ES6+)

## 🚀 Compilação de Códigos C++

Para compilar os arquivos C++, utilize:

```bash
g++ -o programa arquivo.cpp
./programa
```

Ou use a configuração do VS Code disponível em `.vscode/tasks.json`.
